======================================================
Flash Develop Flex 3 AMFPHP Crud Application Generator
======================================================

Generates a FlashDevelop project with all the necessary assets
to start programming an complete Flex 3 application with CRUD operations
over the selected tables/columns using amf protocol.

Requeriments
============
- ALL tables needs to have single column PKs.
- AMFPHP installed of course.

Instructions
============
Modify the values of gen-options.ftl and gen-project.ftl according to your settings.
Right click in the project [FD][Flex3][AmfPHP]CrudAppGen and select "Run Template".
