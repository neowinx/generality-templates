==============================================
SWT/JPA/EclipseLink Crud Application Generator
==============================================

Generates a SWT project with all the necessary assets
to start programming an complete SWT/JPA/EclipseLink application with CRUD operations
over the selected tables/columns.

Requeriments
============
- Eclipse Helios J2EE with WindowBuilder plugin installed.
- ALL tables needs to have single column PKs (NO EXCEPTIONS).

Instructions
============
Create a SWT/JFACE project using the WindowBuilder plugin for eclipse helios. 
Add the Application Client Module and JPA facets. Configure the JPA facet to
use the connection provided in the persistence.xml.
Edit the project.opt files and put the correct values. Rigth click on the template and select Run template option.

Enjoy :D
